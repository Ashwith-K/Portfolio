# Ashwith K — Personal Portfolio

A personal portfolio website built completely from scratch with vanilla HTML, CSS, and JavaScript. This site acts as a responsive, interactive hub to showcase my software development projects, technical skills, academic background, and professional experience.

🔗 **Live Portfolio:** [ashwith-k.github.io/Portfolio](https://ashwith-k.github.io/Portfolio)

---

## 📂 Sections Included

* **Home** — Professional greeting, summary, and quick call-to-action links.
* **About** — Educational timeline, core background, and technical skillset matrix.
* **Experience** — Details regarding my MERN stack development internship.
* **Projects** — Interactive display cards featuring full-stack, cloud, and decentralized applications.
* **Contact** — Direct access channels including Email, LinkedIn, GitHub, and Instagram.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+ Vanilla, framework-free)
* **Typography:** Google Fonts (*Syne* for headings + *DM Sans* for body text)
* **Icons:** Font Awesome Icon Kit
* **Deployment & Hosting:** GitHub Pages
